# **App Name**: Aroma Allure

## Core Features:

- Product Catalog: Display a curated selection of perfumes and decants, with detailed descriptions and images.
- Admin Product Management: Enable administrators to add, edit, and manage products, including images, descriptions, and inventory.
- User Authentication: Secure login for administrators to manage the product catalog.
- Shopping Cart and Checkout: Allow users to add products to a cart, review their order, and proceed to a secure checkout process.
- Product Detail Page: Show comprehensive information about each product, including high-resolution images, descriptions, and customer reviews.
- AI Olfactory Description: Leverage AI to generate a unique and engaging olfactory description for each perfume, acting as a tool to enhance product discovery.

## Style Guidelines:

- Primary color: Vibrant yellow (#F9F506) to evoke luxury and sophistication.
- Background color: Light, desaturated beige (#F8F8F5) for a clean and elegant backdrop.
- Accent color: Analogous golden yellow (#F2EE05) for highlights and interactive elements.
- Body and headline font: 'Spline Sans' (sans-serif) for a modern and readable interface.
- Note: currently only Google Fonts are supported.
- Use Material Symbols for a consistent and modern look.
- Employ a clean and spacious layout, with a focus on high-quality imagery to showcase the products.
- Incorporate subtle animations for transitions and user interactions to enhance engagement.
